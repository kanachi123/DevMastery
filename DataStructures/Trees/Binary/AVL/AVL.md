# ***AVL***

## **Это BST(binary search tree) которое сбалансировано по высоте так,что для каждой его вершины высота её двух поддеревьев различается не более чем на 1**

```
                 5
                / \
|--------------3   7 -| h_right
|-----h_left--/        
|------------2

```
$h_{left}$ = 2{3,2}  
$h_{right}$ = 1{2}  
sub == 2 - 1 = 1

$|h_{left} - h_{right}| \leq 1$  

## ***Balance-Factor(BF)***
## *разница высот левого и правого поддерева может быть*
1. 0
2. 1
3. -1

```
    5
     \
      7
       \
        9
```        
*здесь разница 0 - 2 = |-2| > 1,*
*а если высота больше поиск будет долгим это не оптимально*
*в отличие от BST,AVL делает поворот так чтобы разница была меньше*
*либо равно 1*

*например родитель становится ребенком*
```  
      7
     / \
    5   9
```
*вот еще*
```
      T
     / \
    E   W
   / \
  O   D
 /
P
```
*Левая часть выше правой → нужно правое вращение вокруг T*

*поднимается левый ребёнок*
```
      E
     / \
    O   T
   /
  P
```
*перераспределяем остальные вершины*

*Поддерево D должно остаться справа от E, но слева от T*

*E < D < T*
```
      E
     / \
    O   T
   /   / \
  P   D   W

```

## *Идея поворота*

## *При вращении:*

1. *один узел поднимается вверх*

2. *другой опускается вниз*

3. *среднее поддерево переезжает*

***Но ни один элемент не удаляется***
